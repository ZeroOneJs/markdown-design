# 进阶用法

[Markdown](../component/markdown) 组件集成了 Markdown 实时渲染，目录生成和全文搜索等核心功能，大多数场景下只需引入该组件即可。虽然它功能全面，但在某些特定场景中（如自定义布局等）可能不够灵活。为此，我们提供了单独的 [Render](../component/render)、 [Search](../component/search) 和 [TOC](../component/toc) 组件（Markdown 组件同样基于这三个组件构建）。这样您可以根据实际需求按需引入，自由组合使用。

::: tip 提示
本示例仅展示手动引入组件的方式，如需全局引入[请参考](./quickstart.md#全局引入)。
:::
